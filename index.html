Welcome to pyworldatlas’s Documentation

pyworldatlas is a Python package that compiles detailed data about every country, offering easy-to-use methods for retrieving country profiles, statistics, and more.

Key Features

  • Extensive country data, including official names, population, geography, government details, GDP, and more.

  • Lightweight and efficient, ensuring fast access to country information.

  • No external dependencies, making it easy to install and integrate into any Python project.

Get started by following the instructions below.
